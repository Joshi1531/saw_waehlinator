Auf iPad:

  1. App "Web Inspector" im AppStore laden
  2. Anweisungen in der App folgen
  3. Bei Safari sollte jetzt oben in der Suchzeile ein Puzzleteil sein.
  4. Auf das Puzzleteil tippen -> Web Inspector, dann 'Console' auswählen.
  5. Unten nach dem '>' Zeichen kann man den Code einfügen.
  6. 'Execute' tippen und warten - Fertig.

Auf Tablet

  1. Geht nicht, Pech gehabt

Auf PC

  1. Im Browser rechtsklick auf irgendwas -> Untersuchen
  2. Im neuen Fenster auf 'Console' oder 'Konsole' gehen
  3. Code nach dem '>' Zeichen einfügen
  4. Enter drücken und fertig

Auf Mac

  1. Safari -> Einstellungen -> Erweitert -> Entwickler Menü anzeigen
  2. Oben Entwickler -> Java-Script-Konsole anzeigen ODER 'Option + Command + C'
  3. Auf Konsole gehen
  4. Code einfügen - Fertig
